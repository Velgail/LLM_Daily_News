## こたつLLM速報とは

「こたつLLM速報」（KOTATSU LLM TIMES）は、LLM・SLM・音声LLM・大規模深層学習モデル
など AI 業界の動向を、毎日**こたつから**速報する**ネタ系**のニュースサイトです。

- **発行**: 不定期日刊（編集部の気分次第）
- **発行元**: 個人運営（非営利）
- **執筆**: すべて LLM（大規模言語モデル）が自動生成
- **校閲**: なし（重要）
- **配信**: GitHub Pages（[ソースコード](https://github.com/Velgail/LLM_Daily_News)）

## 取り扱うトピック

| カテゴリ | 内容 |
|---|---|
| LLM | 大規模言語モデル全般。OpenAI / Google / Anthropic / Meta / Mistral など |
| SLM | 小規模言語モデル（Phi / Gemma / Llama 軽量版など） |
| 音声LLM | 音声入出力対応モデル（GPT-4o Voice / Gemini Live など） |
| 大規模DL | 画像生成・動画生成・マルチモーダルなどの大規模深層学習モデル |

## なぜ「こたつ」?

「こたつ記事」（取材を行わず、ネット上の二次情報のみを元に書かれた記事）に由来します。
本紙はその名の通り、**こたつに入った LLM が書いたような** 記事をお届けします。
