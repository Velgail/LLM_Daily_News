### 1. GitHub Copilot、6月1日からAIクレジット制に全面移行——「使い放題」の終焉と開発者コスト管理の新時代

> 「AIコーディングアシスタントは月額定額で使い放題」——その常識が今日から変わる。GitHubが定額制を廃し、トークン消費量に応じた従量課金へ全面移行した。

GitHubは本日2026年6月1日を持って、GitHub Copilotのすべてのプランを**使用量ベースのメーター制課金**へ移行させた。課金単位として「**GitHub AI Credits**」という仮想通貨（1クレジット＝0.01ドル）を導入し、コード補完・次の編集サジェスト以外のCopilot機能（エージェントワークフロー・コードレビュー・高度な推論モデル使用など）はこのクレジットを消費する仕組みに変わる。

各プランの月額料金に含まれるクレジット額は、Copilot Pro+が39ドル分（＝39クレジット）、Copilot Businessが19ドル分、Copilot Enterpriseが39ドル分。含まれる枠を超えた分は追加課金となる。移行直後の3ヶ月間（6月1日〜9月1日）は既存Businessおよびまz Enterprise契約者向けに手厚いクレジット付与が行われる。コード補完・次の編集サジェストは引き続き全有料プランで無制限。

この変更の背景にあるのはAI推論コストの急騰だ。複雑なエージェントセッションでは従来の定額モデルが経済的に成立しなくなった。開発者コミュニティからは「同じ価格で使える量が減る」という不満も出ているが、一方でGitHubは「正確な使用量の可視化が可能になる」というメリットを強調する。Microsoft Build 2026（明日6月2日開幕）でCopilotの新モデルと合わせてさらなる詳細が発表される見込みだ。

### 3. OpenAI Codex、Windows版にComputer Use機能を追加——macOS独占を解除、画面操作でWindows全アプリを制御

> 「AIエージェントはMac専用」——その認識が終わった。CodexがWindowsの画面を見て、クリックして、タイプする。

OpenAIは5月29日、Codexデスクトップアプリ（v26.527）のWindows版アップデートをリリースし、これまでmacOS専用だった「**Computer Use**」機能をWindows 11ユーザーにも提供開始した。Computer Useとは、AIエージェントが画面ピクセルを認識しながらマウスクリック・キーボード入力を行い、ブラウザ・IDE・企業アプリなどあらゆるWindowsアプリケーションを直接操作する機能だ。

今回のアップデートでの主な変更点は3点。まず**Windows Computer Use**——CodexがWindows上のアプリをユーザーの代わりに操作できるようになった。次に**モバイルリモートコントロール**——iPhone/Android上のChatGPTアプリから、遠隔でWindowsマシン上のCodexの進捗確認・作業指示・スレッド管理ができる。そして**Codexプロフィール**——活動履歴・使用統計・トークン使用量を可視化するダッシュボードも追加された。

Computer Use機能はAnthropicのClaude ComputerがMac/Linux向けに提供していた機能に対応するものだ。今回の対応でWindowsユーザー（Codeのデフォルト環境が企業では多数を占める）も本格的なAIエージェント開発・利用が可能になる。翌日のMicrosoft Build 2026でWindows Agent Frameworkが発表されるタイミングと重なることから、AIエージェント実行環境としてのWindows普及を加速させる可能性が高い。
