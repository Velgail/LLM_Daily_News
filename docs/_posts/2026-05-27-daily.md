### 2. OpenAI Codex for Mac が「Appshots」と「ロック時遠隔制御」で進化——AIエージェントが"常時稼働"の相棒へ

> MacのアプリウィンドウをCommand+Commandで瞬時にCodexスレッドへ投影、ロック画面越しにモバイルからもエージェント操作が可能に——開発者体験の大幅な刷新。

OpenAIは5月21日、Codex for Macに2つの注目機能を追加した。ひとつは**Appshots**：Macのキーボードショートカット（Command-Command）を一度押すだけで、現在アクティブなアプリウィンドウのスクリーンショット＋可読テキストをCodexスレッドに瞬時に添付できる機能だ。コードエディタ・ターミナル・ブラウザなど、任意のアプリを「文脈の証拠」としてAIに渡せるため、「このバグを直して」の一言で画面全体のコンテキストが共有される。

もうひとつは**ロック時Mac制御（Locked Computer Use）**：Macがロックされていても、スマートフォンから長時間エージェントタスクを起動・監視できる仕組みだ。Appleの認証プラグインを活用し、厳格な時間的・行動的ガードレールのもとでMacを一時的にアンロック。これにより「就寝中もCodexがコードを書き続け、朝には完成している」というシナリオが現実になる。

5月26日には**ChatGPT Businessも同機能群を取り込んだ大型更新**を実施。Appshots、Goal Mode GA、ブラウザアノテーション改善、共有プラグイン（チームメンバー間でのツール再利用）、Codex利用分析ダッシュボードが一括で企業ユーザーに提供された。AIコーディングエージェントが「呼び出すツール」から「プロアクティブに動く同僚」へと変容しつつある転換点を示す更新だ。
