## 重要なお知らせ

本サイト「**こたつLLM速報**」をご利用になる前に、以下の事項を必ずお読みください。

## 1. 記事はすべて LLM による自動生成です

本サイトに掲載されているすべての記事は、**ネタとして** LLM（大規模言語モデル）が自動生成しています。
人間による編集・校閲・ファクトチェックは行っておりません。

## 2. 内容の正確性は保証されません

LLM は事実と異なる情報（**ハルシネーション**）を出力することがあります。
具体的には、以下のような誤りが含まれている可能性があります。

- 実在しない製品名・モデル名・機能の記載
- 実在する企業・人物に関する事実と異なる記述
- 数値・日付・URL の誤り
- 引用元の取り違え・捏造
- 古い情報・未来の出来事の混在

実在の企業・製品・人物・出来事に関する記述であっても、**そのまま事実として受け取らないでください**。
一次情報は必ず元記事リンク先や公式発表をご確認のうえ、ご自身の判断と責任においてご利用ください。

## 3. 非営利・個人運営です

本サイトは個人が趣味で運営する**非営利**メディアであり、いかなる商業的目的も持ちません。
広告掲載・有料コンテンツ・アフィリエイト等は行っておりません。

## 4. 損害について

本サイトの情報を利用したことにより生じたいかなる損害についても、運営者は一切の責任を負いません。

## 5. 著作権・引用について

- 本サイトの記事中に登場する企業名・製品名・サービス名は、各社の商標または登録商標です。
- 元記事へのリンクは引用目的であり、各記事の著作権は各発行元に帰属します。
- 万一、権利侵害の疑いがある記載を発見された場合は、
  [GitHub Issue](https://github.com/Velgail/LLM_Daily_News/issues) よりご連絡ください。速やかに対応いたします。

## 6. プライバシー

本サイトは GitHub Pages により配信されており、アクセスログ等は GitHub のプライバシーポリシーに従って取り扱われます。
本サイト独自のトラッキング・Cookie 等は設置していません。
